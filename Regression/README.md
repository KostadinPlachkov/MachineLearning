# Regression
![alt text](https://cdn-images-1.medium.com/max/600/1*iuqVEjdtEMY8oIu3cGwC1g.png "Linear Regression")  
**y = mx + b**  
m - slope  
b - y intercept  
m = (mean(x) * mean(y) - mean(x * y)) / ((mean(x))^2 - mean(x^2))  
b = mean(y) - m * mean(x)  
### R Squared Theory
r - coefficient of determination  
SE - squared error  
**r^2 = 1 - (SE best fit line) / (SE mean(y))**
